## Требования

- **Node.js:** версия 18
- **npm:** поставляется вместе с Node.js

---
## Установка и запуск
1. **Склонируйте проект**
   ```
   git clone ...
   ```
2. **Установите зависимости:**
   ```
   npm install
   ```

3. **Сборка проекта для продакшена:**
   ```
   npm run build
   ```


**Запуск в режиме разработки:**

   ```
   npm start
   ```
